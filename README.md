# Cloud Resume Project

## Version 1.1: Day 1 HTML and CSS Foundation

Day 1 focused on building the front end foundation of my cloud resume site using raw HTML and CSS. The goal was to turn an initial layout concept into a clean, functional webpage that could later support cloud based features.

Before writing code, I used Paint.NET to map out the page structure, visual hierarchy, spacing, and overall design direction. This helped me experiment with section placement, content grouping, background imagery, overlays, and how each component would interact on the page before committing anything to the codebase.

The HTML was structured around scalability and organization, with core sections such as the navigation bar, hero section, and project grid built into a clean, logical layout. Each component was designed to be modular so future updates could be added without needing to restructure the entire site.

On the styling side, CSS was used to control layout, spacing, alignment, and visual consistency. Flexbox and CSS Grid were implemented to align elements, organize cards, and maintain a uniform design across sections. Reusable styles were created for panels and cards to keep the design consistent as the project expands.

This version represents the foundation of the site. It demonstrates my ability to plan, structure, and implement a responsive front end using HTML and CSS while applying a deliberate design process from mockup to working webpage.
